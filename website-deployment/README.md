# Website Deployment

This project contains the code and configuration for deploying the Hope Within Universe website. The site is built with React and Vite. It includes:

- Instructions for creating and configuring an S3 bucket or using Netlify/Vercel.
- Steps to enable static website hosting and optionally set up CloudFront for CDN and HTTPS.
- DNS configuration using Route 53 to map the custom domain.

The website will be deployed to the cloud so it can be accessed publicly.
